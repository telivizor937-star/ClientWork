# Telegram Reels Lead Agent

Агент ищет вакансии на монтаж Reels/Shorts/TikTok в публичных Telegram-каналах, собирает удобную CSV-таблицу и готовит короткий уверенный отклик.

Он не пишет людям сам. Это безопаснее для аккаунта: ты получаешь уведомление, открываешь лид с телефона и отправляешь готовый текст вручную.

## Твои правила фильтра

- Берем только Reels / Shorts / TikTok.
- Длинные видео не берем.
- Минимум примерно от 500 ₽ за ролик.
- Постоянка особенно интересна, разовые тоже подходят.
- Тестовое только платное.
- Пропускаем съемку, выезд, офис, без оплаты, SMM, дешевые и смешанные задачи.
- Фокус: только монтаж.

## Файлы

- `channels.txt` - список Telegram-каналов.
- `config.json` - портфолио, минимальная оплата и уведомления.
- `leads.csv` - таблица найденных лидов.
- `last_run.txt` - короткий отчет о последнем запуске.

## Запуск вручную

```powershell
.\run_agent.bat
```

## Автозапуск по расписанию

Запусти:

```powershell
.\install_schedule.bat
```

Он создаст 3 ежедневных запуска в Windows:

- 10:00
- 12:00
- 18:00

## Уведомления в Telegram

Токен бота уже лежит в `config.json`.

Чтобы агент понял, куда присылать лиды:

1. Открой своего бота в Telegram.
2. Напиши ему любое сообщение, например `старт`.
3. Запусти `run_agent.bat` один раз.

Агент сам подхватит `telegram_chat_id` и сохранит его в `config.json`.

Если хочешь прописать вручную, в `config.json` есть поля:

```json
"telegram_bot_token": "ТОКЕН_БОТА",
"telegram_chat_id": "ТВОЙ_CHAT_ID"
```

После этого агент будет присылать уведомление, когда появились новые лиды: ссылку на пост/контакт, бюджет, статус и готовый отклик.
