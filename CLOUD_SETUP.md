# Как запустить агента, чтобы он работал при выключенном компьютере

Самый простой вариант - GitHub Actions. Это облачный запуск по расписанию: компьютер может быть выключен, а агент все равно проверит Telegram-каналы и пришлет новые лиды в бота.

## Что уже настроено

Расписание стоит на время Саратова:

- 10:00
- 12:00
- 18:00

В GitHub это записано как UTC:

- 06:00
- 08:00
- 14:00

## Что нужно сделать один раз

1. Создай новый приватный репозиторий на GitHub.
2. Загрузи туда все файлы из этой папки `tg_reels_lead_agent`.
3. В GitHub открой `Settings` -> `Secrets and variables` -> `Actions`.
4. Добавь два секрета:

`TELEGRAM_BOT_TOKEN`

Токен твоего бота.

`TELEGRAM_CHAT_ID`

Твой chat_id: `2061763455`.

5. Открой вкладку `Actions`.
6. Выбери `Reels lead agent`.
7. Нажми `Run workflow`, чтобы проверить вручную.

После этого агент будет запускаться сам по расписанию.

## Где будет таблица

Файл `leads.csv` будет обновляться прямо в репозитории. Бот будет присылать новые лиды в Telegram: пост/контакт, бюджет и готовый отклик.

## Важно

Не загружай `config.json` с токеном в публичный репозиторий. Лучше создать именно приватный репозиторий и хранить токен только в GitHub Secrets.
